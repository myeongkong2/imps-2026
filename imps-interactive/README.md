# AHA! LAB · IMPS 2026

KLoSA 2024 제10차 자료와 TAN 베이지안 네트워크 분석 결과를 기반으로 만든 정적 연구 결과 페이지입니다.

## 화면 구성

- **확률 직접 보기:** 최종 TAN의 CPT를 이용해 각 노드 상태를 Netica처럼 100% 근거로 고정합니다. 여러 근거를 함께 유지할 수 있고, 노드별 초기화와 기능상태 역방향 추론을 지원합니다.
- **전체 분석 결과:** 표본 특성, AUC, Youden 절단점, 민감도, 특이도, 예측도, 분류표와 24개 노드 상태의 결과를 제공합니다.
- **Netica 사후추론:** 전체 표본으로 추정한 최종 TAN 네트워크의 대표 근거 시나리오와 기능상태 100% 고정 역방향 진단추론을 제공합니다.

## 로컬 실행

`미리보기_실행.bat`를 더블클릭하면 로컬 서버와 페이지가 열립니다. 브라우저 보안 정책상 `index.html`을 직접 여는 대신 로컬 서버를 사용해야 합니다.

## 해석 주의

- R 화면은 10-fold 교차검증의 out-of-fold 예측을 집계한 결과입니다.
- Netica 화면은 전체 표본으로 추정한 최종 TAN의 사후확률입니다.
- 두 화면의 숫자는 역할과 추정 단계가 다르며, 인과효과나 개인의 임상 진단을 의미하지 않습니다.
